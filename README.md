# Cowspotting
## Link -[lily-bel.github.io/cowspotting](https://lily-bel.github.io/cowspotting/)

An app for identifying and cataloging cows in the US.

Features:
- Search through the 104 cow breeds that you can find in the US.
- Identify cows in the wild by answering questions about their features.
- Catalog cows when you spot them and customize their pages with photos and notes on where you found them.
